# LearnSomeThing
Đây là một vài cấu trúc dữ liệu mà tôi đã học được nên tôi đưa lên đây để lưu lại những gì mình học được.
Các cấu trúc dữ liệu này chỉ là phần giới thiệu cơ bản về chúng, còn về các cachs ứng dụng chúng trong thực tế, trong các chương trình lớn bạn cần tìm hiểu thêm ở một số trang web khác uy tín hơn.
